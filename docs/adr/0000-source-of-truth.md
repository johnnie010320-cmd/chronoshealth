# ADR 0000 — 작업 절차서를 1차 근거로 채택

- **Status**: Accepted
- **Date**: 2026-05-12
- **Decider**: Johnnie

## Context

Chronos Health 프로젝트는 의료·금융 규제·Web3·ML 등 다영역이 결합되어 결정 사항이 많다. 결정의 정합성을 보장하려면 단일 정본(source of truth)이 필요하다.

## Decision

`docs/work-procedure.txt` (크로노스헬스 작업 절차서 v1.0, 2026-05-12 작성) 를 본 프로젝트의 1차 근거 문서로 채택한다.

- 모든 ADR, 명세, CLAUDE.md, 서브에이전트, 코드는 본 절차서를 위배할 수 없다.
- 절차서와 충돌하는 기존 설정은 절차서 기준으로 즉시 갱신한다.
- 절차서를 변경할 때는 새 ADR로 변경 사유 + 영향 범위를 기록한 후 갱신한다.

## Consequences

**긍정**
- 결정 정합성 보장. "어디 기준이냐" 논쟁 차단.
- 신규 합류자가 절차서 1개 + ADR 목록만 읽으면 컨텍스트 진입 가능.

**부정 / 위험**
- 절차서 자체에 오류가 있을 경우 전파 위험 → 분기별 검토 ADR로 보정.

## Alternatives Considered

- CLAUDE.md를 1차 근거로 채택 (rejected: 자동 로드되지만 정책·운영·로드맵 등 장기 문서를 담기에는 짧고 단호한 톤을 유지하기 어려움)
- 외부 위키(Notion 등)를 1차 근거로 채택 (rejected: 재현 가능성 / 깃 추적 / 감사 추적성 부족)
