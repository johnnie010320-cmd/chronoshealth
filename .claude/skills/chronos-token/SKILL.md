# Chronos Token (CHRO) Skill

> 근거: `docs/work-procedure.txt` 8장 + ADR 0004 (체인 선택)

## 토큰 사양 (초안, 상장 직전 확정)

| 항목 | 값 |
|------|---|
| 심볼 | **CHRO** |
| 표준 | ERC-20 + Permit + Votes |
| 체인 | **Polygon PoS 1차** (Base / Arbitrum 확장 검토) |
| 총 발행량 | 1,000,000,000 (10억, 고정) |
| 인플레이션 | 없음 |

| 카테고리 | 비율 | 비고 |

베스팅: Sablier 또는 자체 컨트랙트.

## Health-to-Earn 정산 (절차서 8.2)

```
기여 점수 = 데이터 신선도 × 품질 × 동의 범위
```

- **주기**: 주간 정산
- **방식**: 머클 드롭 — 사용자가 직접 클레임 트랜잭션 (가스 효율)
- **클레임 한도**: 사용자별 일일 클레임 횟수 제한
- **시빌 방지** (3중):
  1. 디바이스 어테스테이션 (Apple App Attest / Google Play Integrity)
  2. 단순 KYC 등급제 (등급별 보상 한도)
  3. 영지식 건강 증명 (Circom Groth16)

## 효용 (절차서 8.1)

1. **데이터 보상** — Health-to-Earn
2. **프리미엄 결제** — 고도화 리포트, 1:1 코칭, API 라이선스
3. **거버넌스** — 보상 비율 / 신규 데이터 카테고리 / 파트너십
4. **파트너십 라이선스 결제** — 보험사·연구기관 익명 통계

소각: 프리미엄 결제 수익의 20%를 분기마다 소각 (온체인 로그).

## 거버넌스 (절차서 8.3)

- Snapshot 오프체인 투표 → 통과 시 온체인 타임락 큐잉 → 48h 후 실행
- 정족수: 발행량의 4% 참여 + 단순 과반
- 핵심 항목(컨트랙트 업그레이드): 60% 찬성
- 위임(Delegation) 허용 / 베스팅 중 토큰도 의결권 행사 가능

## 운영 보안 (절차서 8.4)

- **Gnosis Safe 3-of-5** (Ledger 하드웨어 지갑 + 지리적 분산)
- **OpenZeppelin Defender Sentinel**: 비정상 대량 이체, 권한 변경, 가스 급등 알림
- **Tenderly**: 24/7 모니터링
- **타임락**: 48시간 (업그레이드 / 권한 변경)
- **버그 바운티**: Immunefi 등록, 상한 $100K 시작

## 외부 감사 (절차서 5.2 P5)

- CertiK 또는 Trail of Bits
- 통과 기준: 고위험·치명적 결함 0건
- 감사 패키지: `/audit-prep` 슬래시 커맨드 자동 생성
- 보고서 공개 (LBank 상장 신청 자료)

## 보안 체크리스트 (감사 전 자체 점검)

## 컴플라이언스 (절차서 7.4)

- **유틸리티 토큰 의견서**: 한국 + 발행 법인 관할(케이맨/싱가포르) 로펌에서 모두 수령
- **KYC/AML**: 토큰 수령 한도 + 익명 사용자 상한선. 상장 전부터 적용.
- **백서/토크노믹스**: 감사 신청 전 v1.0 공개. 변경은 거버넌스 투표 + 공시.

## 거래소 / 마켓 메이커

- LBank 또는 동급 1~2곳 상장 (절차서 5.2 P6)
- 마켓 메이커 3개월 계약 (DWF Labs / Wintermute 등)
- 운영 지표(MAU·리포트 사용)와 토큰 지표 별도 대시보드 (가격 변동에 휘둘리지 않도록)

## 금지

- private key 코드 / `.env` 커밋
- 컨트랙트 owner 권한 EOA 단독 보유 (반드시 멀티시그)
- 보상 풀 무한 발행 (반드시 cap)
- 사용자 데이터 raw를 온체인에 기록 (메타 해시만)
