## 제1조 (총칙)
Chronos Health(이하 "회사")는 「개인정보 보호법」, 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」, GDPR(EU 일반 데이터 보호 규정)을 준수하며, 회원의 개인정보 보호를 최우선으로 합니다.

## 제2조 (수집하는 개인정보 항목)

### 1. 식별정보 (identity-vault 저장)
- 이름, 이메일, 전화번호, 출생연도, 성별, 국적
- 비밀번호 (PBKDF2-SHA256 해싱 + 16-byte 솔트, 평문 미저장)

### 2. 분석정보 (analysis DB 저장, 가명화)
- 건강 설문 23문항 응답 (신체측정값·생활습관·가족력)
- 예측 리포트 (생체 나이·5년 위험)
- 루틴 기록 (음식·운동·수면)
- 커뮤니티 게시·댓글·좋아요
- 포인트 적립·사용 내역

### 3. 자동 수집
- 접속 IP의 HMAC 해시 (분쟁 대응 용)
- 사용자 에이전트(User-Agent)
- 접속 시각

## 제3조 (개인정보의 처리 목적)
회사는 수집한 개인정보를 다음 목적으로만 사용합니다.

1. **서비스 제공** — 회원 식별, 예측 리포트 산출, 활력 점수 계산, 루틴 기록 보관, 커뮤니티 운영
2. **계정 보안** — 본인 인증, 부정 가입 방지, 비밀번호 검증
3. **고객 지원** — 문의 응대, 분쟁 해결
4. **통계 분석** — 가명화된 데이터로 서비스 개선 (개별 회원 식별 불가)
5. **법적 의무 이행** — 관계 법령에 따른 보존, 수사 협조 등

**다음 목적으로는 절대 사용하지 않습니다.**
- 의료 진단·처방·치료
- 보험사·은행 등 제3자 평가 자료 제공
- 마케팅 자동화 (별도 동의 없이)

## 제4조 (개인정보의 격리 구조)

회사는 PII 격리 원칙(ADR 0003)에 따라 다음 구조로 데이터를 격리합니다.

```
identity-vault (chronoshealth-identity D1)
├── 회원 식별정보 (이름·이메일·전화·국적·생년·성별)
├── 비밀번호 해시
└── 세션 토큰

analysis DB (chronoshealth-analysis D1)
├── user_pseudonym_id (가명화 식별자, UUID)
├── 설문 응답 / 예측 리포트 (PII 0)
├── 루틴 / 커뮤니티 / 포인트 (PII 0)
└── 베타 등록자 이메일은 HMAC-SHA256 가명화 저장
```

식별정보와 분석정보는 **별도 D1 인스턴스**에 저장되며, 분석정보 측에서는 가명화된 ID(`user_pseudonym_id`)만 사용합니다. 두 데이터를 결합하는 권한은 회사 내부 관리자(감사 로그 기록 대상)만 보유합니다.

## 제5조 (보유 및 이용 기간)

| 데이터 종류 | 보유 기간 | 비고 |
|------------|----------|------|
| 식별정보 | 회원 탈퇴 시까지 | 탈퇴 즉시 삭제 |
| 분석정보 (가명화) | 탈퇴 후 최대 3년 | 통계·연구 목적 (개별 식별 불가) |
| 동의 이력 | 탈퇴 후 5년 | 「개인정보 보호법」 §15-4 |
| 접속 로그 | 3개월 | 「통신비밀보호법」 |

회원이 명시적으로 즉시 삭제를 요청하면 분석정보도 즉시 삭제됩니다.

## 제6조 (제3자 제공 및 위탁)

### 1. 제3자 제공
회사는 회원의 개인정보를 제3자에게 제공하지 않습니다. 단, 다음 경우는 예외입니다.
- 회원의 사전 동의가 있는 경우
- 법령에 의거 또는 수사기관의 적법한 절차에 의한 요구가 있는 경우

### 2. 위탁
회사는 다음 업체에 인프라 운영을 위탁합니다.
- **Cloudflare, Inc.** — 데이터 호스팅 (Workers, D1, R2, Pages)
- 위탁 처리 항목: 식별정보 + 분석정보 일체
- 위탁 기간: 회원 탈퇴 시까지

## 제7조 (국외 이전)

회사는 Cloudflare 글로벌 네트워크를 통해 데이터를 처리하며, 데이터는 다음 지역에 분산 저장됩니다.
- **저장 리전**: Cloudflare D1 (전 세계 자동 복제), R2 (특정 리전 지정 가능)
- **이전 사유**: 글로벌 안정성, 응답 지연 단축
- **보호 조치**: TLS 1.3 암호화 전송, AES-256 저장, ISO 27001/SOC 2 인증

회원은 동의를 거부할 권리가 있으며, 거부 시 서비스 이용이 제한될 수 있습니다.

## 제8조 (정보주체의 권리)

회원은 다음 권리를 행사할 수 있습니다.
1. **열람** — 본인 식별정보·분석정보 일체 조회
2. **정정** — 잘못된 정보 수정
3. **삭제** — 회원 탈퇴 또는 특정 데이터 삭제 요청
4. **처리 정지** — 동의 철회
5. **이동** — JSON 형식으로 본인 데이터 일괄 내보내기

권리 행사 방법: 회원 메뉴 → 마이페이지 → 정보 수정 / 탈퇴

회사는 권리 행사 요청을 받은 후 **10영업일 이내** 처리합니다.

## 제9조 (개인정보의 안전성 확보 조치)

회사는 개인정보를 보호하기 위해 다음 조치를 시행합니다.

1. **기술적 조치**
   - 비밀번호 PBKDF2-SHA256 일방향 해싱 + 솔트
   - 전송 구간 TLS 1.3 암호화
   - 저장 구간 AES-256 암호화 (Cloudflare 인프라 기본 제공)
   - 베타 등록 이메일 HMAC-SHA256 가명화

2. **관리적 조치**
   - PII 격리 원칙 (ADR 0003) 강제
   - 관리자 접근 시 감사 로그 기록 (계획)
   - 비밀번호 시도 횟수 제한 (분당 5회)

3. **물리적 조치**
   - Cloudflare 데이터센터 24/7 모니터링, 출입 제한

## 제10조 (쿠키 및 자동 수집 도구)

회사는 다음 목적으로 브라우저 저장소를 사용합니다.

| 저장소 | 항목 | 목적 | 유지 기간 |
|--------|------|------|----------|
| localStorage | `chronos.locale` | 언어 설정 | 영구 (사용자 삭제 시까지) |
| localStorage | 세션 토큰 | 로그인 유지 | 30일 |
| localStorage | 공지 dismiss 상태 | UX | 사용자 삭제 시까지 |

회사는 **광고성 쿠키(Google Analytics, Facebook Pixel 등)를 사용하지 않습니다**.

## 제11조 (아동의 개인정보)

회사는 **만 19세 미만의 회원 가입을 차단**합니다. 회원 가입 시 출생연도 검증을 통해 만 19세 미만으로 확인되면 가입 절차가 즉시 종료됩니다.

만일 만 19세 미만 아동의 개인정보가 수집된 사실이 확인되면, 회사는 즉시 해당 데이터를 삭제하고 보호자에게 통지합니다.

## 제13조 (위기·위급 상황 안내)

회원이 입력한 자가 인식 항목에서 자살·심각한 정신건강 위기 신호가 감지되면, 회사는 다음 핫라인을 노출합니다.
- **자살예방상담전화 1393** (24시간)
- **정신건강위기상담전화 1577-0199** (24시간)

본 안내는 의료 행위가 아니며, 응급 상황 시에는 즉시 119 또는 가까운 응급실로 연락해 주십시오.
